### 描述

- 该接口提供版本：v9.9.9+。
- 该接口所需权限：负载均衡删除。
- 该接口功能描述：业务下批量删除负载均衡。有监听器的负载均衡不能删除。

### URL

DELETE /api/v1/cloud/bizs/{bk_biz_id}/load_balancers/batch

### 输入参数

| 参数名称      | 参数类型         | 必选 | 描述        |
|-----------|--------------|----|-----------|
| bk_biz_id | int64        | 是  | 业务ID      |
| ids       | string array | 是  | 负载均衡的ID列表 |

### 调用示例

```json
{
  "ids": [
    "00000001",
    "00000002"
  ]
}
```

### 响应示例

```json
{
  "code": 0,
  "message": "ok"
}
```

### 响应参数说明

| 参数名称    | 参数类型   | 描述   |
|---------|--------|------|
| code    | int32  | 状态码  |
| message | string | 请求信息 |
