### 描述

- 该接口提供版本：v1.0.0+。
- 该接口所需权限：IaaS资源操作。
- 该接口功能描述：批量关闭虚拟机。（需注意采用关机方式为强制关机）

### URL

POST /api/v1/cloud/cvms/batch/stop

### 输入参数

| 参数名称 | 参数类型         | 必选  | 描述       |
|------|--------------|-----|----------|
| ids  | string array | 是   | 虚拟机的ID列表 |

### 调用示例

```json
{
  "ids": [
    "00000001",
    "00000002"
  ]
}
```

### 响应示例

```json
{
  "code": 0,
  "message": "ok"
}
```

### 响应参数说明

| 参数名称    | 参数类型   | 描述   |
|---------|--------|------|
| code    | int32  | 状态码  |
| message | string | 请求信息 |
