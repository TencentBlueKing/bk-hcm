### 描述

- 该接口提供版本： v1.1.17+。
- 该接口所需权限：IaaS资源创建。
- 该接口功能描述：创建VPC。

### URL

POST /api/v1/cloud/vpcs/create

### 输入参数

#### tcloud

| 参数名称        | 参数类型   | 必选 | 描述                |
|-------------|--------|----|-------------------|
| account_id  | string | 是  | 账号ID              |
| region      | string | 是  | 地域                |
| name        | string | 是  | 名称                |
| ipv4_cidr   | string | 是  | IPv4 CIDR         |
| bk_cloud_id | int64  | 是  | 云区域ID，-1表示没有绑定云区域 |
| subnet      | object | 是  | 子网                |
| memo        | string | 否  | 备注                |

##### subnet

| 参数名称      | 参数类型    | 必选  | 描述        |
|-----------|---------|-----|-----------|
| name      | string  | 是   | 子网名称      |
| ipv4_cidr | string  | 是   | IPv4 CIDR |
| zone      | string  | 是   | 可用区       |

```json

  "code": 0,
```

| 参数名称    | 参数类型   | 描述   |
|---------|--------|------|
| code    | int32  | 状态码  |
| message | string | 请求信息 |
| data    | object | 响应数据 |
#### data

| 参数名称 | 参数类型   | 描述   |
|------|--------|------|
| id   | string | 单据ID |
