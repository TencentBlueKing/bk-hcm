### 描述

- 该接口提供版本：v1.0.0+。
- 该接口所需权限：IaaS资源操作。
- 该接口功能描述：卸载云盘。

### URL

POST /api/v1/cloud/disks/detach

### 输入参数

| 参数名称         | 参数类型    | 必选  | 描述                           |
|--------------|---------|-----|------------------------------|
| disk_id      | string  | 是   | 云盘 ID                        |

### 调用示例

如卸载云盘 ID 是00000050的云盘。

```json
{
    "disk_id": "00000050"
}
```

### 响应示例

```json
{
  "code": 0,
  "message": "ok"
}
```

### 响应参数说明

| 参数名称      | 参数类型   | 描述   |
|-----------|--------|------|
| code      | int    | 状态码  |
| message   | string | 请求信息 |
