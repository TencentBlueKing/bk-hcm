### 描述

- 该接口提供版本：v1.2.1+。
- 该接口所需权限：业务访问。
- 该接口功能描述：删除收藏的业务。

### URL

DELETE /api/v1/cloud/bizs/{bk_biz_id}/collections/bizs

### 输入参数

### 调用示例

#### 获取详细信息请求参数示例

```json
{
  "bk_biz_id": 1
}
```

```json
{
  "code": 0,
  "message": ""
}
```
