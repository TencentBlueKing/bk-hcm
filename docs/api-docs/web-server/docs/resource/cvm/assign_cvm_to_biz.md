### 描述

- 该接口提供版本：v1.8.0+。
- 该接口所需权限：资源分配。
- 该接口功能描述：分配虚拟机到业务下。

### URL

POST /api/v1/cloud/cvms/assign/bizs

### 输入参数

| 参数名称      | 参数类型         | 必选  | 描述       |
|-----------|--------------|-----|----------|
| cvms   | object array | 是   | 分配的虚拟机列表 |

#### cvms
| 参数名称        | 参数类型   | 必选 | 描述      |
|-------------|--------|----|---------|
| cvm_id      | string | 是  | 虚拟机的ID  |
| bk_biz_id   | int64  | 是  | 业务ID    |
| bk_cloud_id | int64  | 是  | 管控区域ID  |

### 调用示例

```json
{
  "cvms": [
    {
      "cvm_id": "00000001",
      "bk_biz_id": 1,
      "bk_cloud_id": 1
    },
    {
      "cvm_id": "00000002",
      "bk_biz_id": 1,
      "bk_cloud_id": 1
    }
  ]
}
```

### 响应示例

```json
{
  "code": 0,
  "message": "ok"
}
```

### 响应参数说明

| 参数名称    | 参数类型   | 描述   |
|---------|--------|------|
| code    | int32  | 状态码  |
| message | string | 请求信息 |
