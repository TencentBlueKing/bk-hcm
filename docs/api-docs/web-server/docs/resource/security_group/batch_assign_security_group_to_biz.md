### 描述

- 该接口提供版本：v1.8.0+。
- 该接口所需权限：资源-资源分配。
- 该接口功能描述：批量分配安全组到业务，安全组会被默认分配到管理业务下。

### URL

POST /api/v1/cloud/security_groups/assign/bizs/batch

### 输入参数

| 参数名称   | 参数类型         | 必选 | 描述                 |
|--------|--------------|----|--------------------|
| sg_ids | string array | 是  | 需要分配的安全组ID列表，最大100 |

### 调用示例

```json
{
  "sg_ids": [
    "00000001",
    "00000002"
  ]
}
```

### 响应示例

```json
{
  "code": 0,
  "message": "ok"
}
```

### 响应参数说明

| 参数名称    | 参数类型   | 描述   |
|---------|--------|------|
| code    | int32  | 状态码  |
| message | string | 请求信息 |
