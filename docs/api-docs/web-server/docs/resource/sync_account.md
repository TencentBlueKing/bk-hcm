### 描述

- 该接口提供版本：v1.0.0+。
- 该接口所需权限：账号编辑。
- 该接口功能描述：同步账号下所有HCM纳管资源。

### URL

POST /api/v1/cloud/accounts/{account_id}/sync

### 输入参数

| 参数名称       | 参数类型   | 必选  | 描述   |
|------------|--------|-----|------|
| account_id | string | 是   | 账号ID |

### 调用示例

```json
```

### 响应示例

```json
{
  "code": 0,
  "message": "ok"
}
```

### 响应参数说明

| 参数名称    | 参数类型   | 描述   |
|---------|--------|------|
| code    | int32  | 状态码  |
| message | string | 请求信息 |
