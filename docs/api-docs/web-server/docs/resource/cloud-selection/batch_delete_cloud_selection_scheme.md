### 描述

- 该接口提供版本：v9.9.9+。
- 该接口所需权限：资源选型-方案删除。
- 该接口功能描述：批量删除云选型方案。

### URL

DELETE /api/v1/cloud/selections/schemes/batch

### 输入参数

| 参数名称 | 参数类型         | 必选 | 描述     |
|------|--------------|----|--------|
| ids  | string array | 是  | 方案ID列表 |

### 调用示例

```json
{
  "ids": [
    "00000001",
    "00000002"
  ]
}
```

### 响应示例

```json
{
  "code": 0,
  "message": "ok"
}
```

### 响应参数说明

| 参数名称    | 参数类型   | 描述   |
|---------|--------|------|
| code    | int32  | 状态码  |
| message | string | 请求信息 |
